Quicksort Algorithm
The quicksort algorithm is a divide-and-conquer algorithm that works by selecting a 'pivot' element from the array and partitioning the other elements into two sub-arrays, according to whether they are less than or greater than the pivot. The sub-arrays are then recursively sorted.

Python Implementation
def quicksort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quicksort(left) + middle + quicksort(right)

◈ Example usage

arr = [3,6,8,10,1,2,1]
print("Original array:", arr)
print("Sorted array:", quicksort(arr))

How it Works

1.If the length of the array is 1 or less, return the array (since it's already sorted)

2.Select a pivot element from the array

3.Create three lists: left, middle, and right

4.Iterate through the array and append elements to the corresponding list based on their value relative to the pivot

5.Recursively call the quicksort function on the left and right lists

6.Concatenate the results of the recursive calls with the middle list to produce the final sorted array


Time Complexity
The average time complexity of quicksort is O(n log n), making it suitable for large datasets. However, in the worst-case scenario (e.g., when the pivot is always the smallest or largest element), the time complexity can be O(n^2).

def execute_python_code(code):

    """

        with tempfile.NamedTemporaryFile(delete=False, suffix=".py", mode="w") as f:

        # Run the Python file with a timeout of 10 seconds
        result = subprocess.run(

            timeout=10
        )

            return f"Error (Return Code {result.returncode}):\n{result.stderr}"

        # Return the output
        return f"Output:\n{result.stdout}"

    except Exception as e:
        # Handle any exceptions that may occur
        return f"An error occurred: {str(e)}"

    finally:
        # Remove the temporary file
        if os.path.exists(temp_file):
